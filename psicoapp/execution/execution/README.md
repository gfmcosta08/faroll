# Execution (Camada 3 – Fazer o trabalho)

Scripts **determinísticos** em Python que executam a lógica de negócio.

- Variáveis de ambiente, tokens de API etc. ficam no `.env`
- Chamadas de API, processamento de dados, arquivos e banco vão aqui
- Código confiável, testável e bem comentado

**Antes de escrever um novo script**, verifique se já existe algo em `execution/` que atenda à diretiva. Só crie novos scripts quando realmente não houver alternativa.

Quando algo quebrar: corrija o script, teste, e atualize a diretiva correspondente com o aprendizado (self-annealing).
